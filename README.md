### 项目说明
挑战6秒是精典的小游戏，通过控制按键来使时钟精准停在6秒整。  
本程序仅仅为了学习的目的，还存在很多不完善甚至错误的地方，暂时不去修改了，请谨慎参考。  
特别感谢@**简单（2864292458@qq.com）同学**，本程序即以他的[“挑战六秒”](https://codebus.cn/luoyh/challenge)程序为基础，添加了部分注释，进行了适当修改并添加了部分功能。
